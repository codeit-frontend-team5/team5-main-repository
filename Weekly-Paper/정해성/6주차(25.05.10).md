## 예시의 코드를 실행할 때, 콘솔에 출력될 값과 그 이유를 설명해 주세요.

```js
// 1번
let num = 1

// 2번
setTimeout(() => {
  num = 2
}, 0)

// 3번
num = 3

// 4번
console.log(num)
```

### 출력값

```
3
```

### 이유

- `setTimeout(..., 0)`은 **비동기 콜백 큐에 등록**되며, 지연 시간이 0이어도 **즉시 실행되지는 않음**
- 자바스크립트는 **동기 코드**를 먼저 실행한 후, **이벤트 루프를 통해 비동기 코드**를 나중에 실행함
- `num = 3`이 먼저 실행되므로, `console.log(num)` 시점에서는 **`num`이 3**

## 🇶 AJAX에 대해 설명해 주세요.

### 정의

> AJAX는 **Asynchronous JavaScript and XML**의 약자로,  
> 웹 페이지 전체를 새로 고치지 않고,  
> 자바스크립트를 이용해 **서버와 비동기적으로 데이터를 주고받는 기술**입니다.

### 주요 특징

- 전체 페이지 리로드 없이 필요한 데이터만 받아올 수 있음
- 초창기에는 XML을 사용했으나, 지금은 **JSON**이 주로 쓰임
- 대표적인 구현 방법:
  - `XMLHttpRequest`
  - `fetch API`
  - `axios` (라이브러리)

### 예시

- 댓글 입력 후 새로고침 없이 댓글이 추가되는 기능
- 무한 스크롤 구현 시, 스크롤 하단 도달 시 서버에서 추가 데이터 로딩
