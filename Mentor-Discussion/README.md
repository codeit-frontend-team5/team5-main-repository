### Pull Request Merge

- Merge
  > 머지 커밋을 생성합니다. 해당 머지커밋에는 Pull을 진행하며 변경된 커밋내역을 들이 포함됩니다.
- Squash Merge
  > 브랜치의 커밋을 하나로 통합하여 머지합니다.
- Rebase & Merge
  > 베이스 브랜치를 변경하고 변경된 헤드 기반으로 머지합니다.

스쿼시하면 이전 헤드는 사라집니다.
